# calculator
InternSim: Calculator (C++)
🚀 Features
Basic Arithmetic: Addition, Subtraction, Multiplication, and Division.

Continuous Calculation: Perform multiple operations without restarting the app.

Input Validation: Validates numeric inputs and prevents crashes on division by zero.

Simple Interface: Clean and intuitive CLI feedback.

🛠️ Requirements
To compile and run this project, you will need:

A C++ compiler (e.g., g++ via MinGW or Clang).

Standard C++ libraries (included in most compilers).
